Deno

A Scripting Host for JavaScript and TypeScript

Introduction

Differences to Node.js:

  • ES6 modules instead of CommonJS modules
  • Improved module distribution
  • No centralized module repository
  • Integral security concept, secure by default
  • Permissions have to be granted explicitly
  • Integrated Tpypescript compiler
  • Integrated unit testing, code formatting, linting, dependency checking and bundling
  • File change watcher, reloads changed modules automatically
